# PokeParadise

Trabajo Práctico Obligatorio 1 — Front End.

## Estructura de layout (clon de tiendadecafe.com.ar)

1. Barra superior (top-bar) con info de envíos y links rápidos
2. Header sticky: logo + nav horizontal + íconos + Mi Cuenta
3. Hero de ancho completo con imagen de fondo y CTA
4. Sección de categorías (4 cards)
5. Grilla de productos destacados (4 cards tipo e-commerce)
6. Banner promocional full-width
7. Sección newsletter (suscripción por email)
8. Footer 4 columnas con logo, nav, empresa y contacto

## Responsividad

4 breakpoints según enunciado:
- Tablet horizontal: `max-width: 1080px`
- Tablet vertical: `max-width: 768px`
- Móvil 1: `max-width: 480px`
- Móvil 2: `max-width: 375px`
